My First Tutorial Ever

One year, and one month later...

Posted by on February 14, 2019

Here you can find a tutorial that I wrote as an introduction to some of the fundamental concepts for machine learning.

I presented it this way because, as an interactive learner, I think Jupyter notebooks are a great way to learn a new concept. GitHub also makes it great by allowing you to run them in-browser.

The content from this tutorial was all knowledge that I learned from Andrew Ng's Coursera course, which I highly recommend.

As always, I welcome any feedback on this tutorial and any recommendations for further topics.
